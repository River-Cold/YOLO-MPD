### Installation
Clone repo and install requirements.txt in a Python>=3.8.0 conda environment, including PyTorch>=1.12.
```
git clone https://github.com/River-Cold/YOLO-MPD.git
cd YOLO-MPD
pip install -r requirements.txt
```
